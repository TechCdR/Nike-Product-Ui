### Run Locally

To run **Nike-Product-Ui** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/TechCdR/Nike-Product-Ui.git
```

Windows:

```bash
git clone https://github.com/TechCdR/Nike-Product-Ui.git
```
